# klipi

a lightweight macos clipboard history manager.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## features

- press `alt+cmd+.` to open the clipboard history popup
- use arrow keys to navigate, enter to copy
- click on an item to copy it directly
- lightweight and fast

## installation

### homebrew

```bash
brew install --cask prolifel/homebrew-tap/klipi
```

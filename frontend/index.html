Klipi

Clipboard History,
Made Simple

A lightweight macOS clipboard history manager that lives in your menu bar. Fast, minimal, and stays out of your way.

Install with Homebrew View on GitHub

Features

Everything you need in a clipboard manager, nothing you don't.

⚡

Lightning Fast

Opens instantly with a keyboard shortcut. No lag, no delays.

📋

50 Item History

Stores your last 50 clipboard items, persisted across app restarts.

🔍

Quick Search

Find any clipboard item quickly with built-in search.

⌨️

Keyboard Navigation

Navigate with arrow keys, copy with Enter. No mouse needed.

🎨

Menu Bar Native

Lives in your menu bar, always accessible, never in the way.

🔧

Customizable

Set your own keyboard shortcut. Make it yours.

Installation

Get Klipi running in seconds with Homebrew.

Install via Homebrew (recommended):

brew tap prolifel/homebrew-tap && brew install klipi

How to Use

Simple and intuitive. Here's how to get started.

1

Open Klipi

Press Alt + Cmd + . or click the menu bar icon.

2

Grant Permissions

Allow Input Monitoring access when prompted for global keyboard shortcuts.

3

Browse History

Use ↑ ↓ arrow keys to navigate your clipboard history.

4

Copy an Item

Press Enter or click an item to copy it to your clipboard.

5

Search (Optional)

Start typing to filter clipboard items by content.
